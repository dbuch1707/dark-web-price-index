# Dashboard Color Palette

## Background & Text
| Element | Hex | Usage |
|---------|-----|-------|
| Dashboard background | `#0A0A0A` | All sheet and dashboard backgrounds |
| Primary text | `#FFFFFF` | Headlines, bar labels, KPI values |
| Secondary text | `#888888` | Subtitles, axis labels, subheadlines |
| Tertiary text | `#555555` | Footer, metadata, source lines |
| Monospace text | `#555555` | Footer font (Courier New) |

## Threat Level Colors
| Level | Hex | Price Range |
|-------|-----|-------------|
| Critical | `#C0392B` | $10,000+ |
| High | `#E67E22` | $500–$9,999 |
| Medium | `#F1C40F` | $50–$499 |
| Low | `#95A5A6` | Under $50 |

## KPI Values
| KPI | Color | Hex |
|-----|-------|-----|
| $1 (SSN) | Red | `#C0392B` |
| $20–$100 (Fullz) | Amber | `#E67E22` |
| $200K+ (Zero-day) | Light gray | `#AAAAAA` |

## Map Palette (Sequential Red)
| Stop | Hex | Meaning |
|------|-----|---------|
| Low end | `#FFCCCC` | Low price / low data |
| High end | `#7B0000` | High price / high targeting |
| No data | `#1A1A1A` | Country not in dataset |
